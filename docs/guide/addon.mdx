# ➕ AddOn

## 简介
* Microsoft Translate
  * Microsoft Translate 为免费翻译器，可以直连使用，Token有效期为十分钟
  * 启用后，每十分钟自动更新并填写Token，同时会将翻译器自动切换为Microsoft Translate（以免除tvOS用户操作）
* Akamaized
  * 提供 Akamaized.net 流媒体解决方案的字幕翻译

## 使用说明
* Microsoft Translate
  * 安装后，等待每10分钟自动运行，首次使用需等待10分钟第一次刷新
* Akamaized
  * 详见[🍿 DualSubs: 🔣 Universal#使用说明](./universal#使用说明)

## 功能列表
* Microsoft Translate
  * 每十分钟自动更新并填写Token
  * 将翻译器自动切换为Microsoft Translate
* Akamaized
  * 对于 `https://*.akamaized.net/out/v1/${hash}.m3u8` 的直播流地址进行字幕翻译
